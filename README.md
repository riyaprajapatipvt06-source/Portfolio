# 🎨 Riya Prajapati | UX Design Portfolio

A modern, responsive, and animated personal portfolio website built to showcase my journey, projects, and skills as a UX Design student. 

## 📖 About This Project

This portfolio is designed with a vibrant "Amoeba Organic" color palette, featuring glassmorphism elements, smooth scroll behaviors, and interactive hover effects. It serves as a digital resume and project showcase for my B.Des (UX Design) coursework at DIT University.

## ✨ Features

* **Custom UI/UX:** A unique organic design with floating blob animations and soft gradients.
* **Fully Responsive:** Adapts seamlessly to mobile, tablet, and desktop screens using CSS Grid and Flexbox.
* **Interactive Image Gallery:** Project images feature a zoom-in hover effect and open in a full-screen modal when clicked.
* **Sticky Glassmorphism Navbar:** A floating, blurred navigation bar that stays accessible as you scroll.
* **Smooth Animations:** Engaging CSS keyframe animations for page loads, button hovers, and structural elements.

## 🗂️ Portfolio Sections

1.  **Home:** A welcoming hero section with my bio, animated profile photo, and quick call-to-action buttons.
2.  **About Me:** Details about my educational background (DIT University, SGRR) and my passion for design.
3.  **My Work:** A showcase of my academic projects, including:
    * Elements of Design
    * History and Art of Evolution
    * Sketching and Drawing
    * Visualising and Story Telling
    * Typography
4.  **Skills:** My toolset categorized into Technical (HTML, CSS, JS), Design (Photoshop, Color Theory, etc.), and Soft Skills.
5.  **Contact:** Easy ways to get in touch via email and phone.

## 🛠️ Technologies Used

* **HTML5:** Semantic structure and layout.
* **CSS3:** Custom CSS variables, animations, media queries, and styling.
* **JavaScript:** Vanilla JS for handling the image expansion modal.
* **Google Fonts:** Space Grotesk, Syne, Playfair Display, and Poppins for strong typographic hierarchy.
* **FontAwesome:** Scalable vector icons for the contact section and buttons.
